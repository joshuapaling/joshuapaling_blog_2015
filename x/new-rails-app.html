Stuff to do / remember when setting up a new rails app

  • Use spring, and springify binstubs
bundle install
bundle exec spring binstub --all

On spring's advice, I've installed direnv so also add a .envrc file in the project root, with the content:

PATH_add bin

Use bundle outdated to check for outdated gems. See http://stackoverflow.com/questions/6985945/bundler-how-to-check-if-gems-are-up-to-date and http://stackoverflow.com/questions/4696932/can-bundler-show-me-which-gems-in-gemfile-have-newer-versions-eg-dry-run-of-bu
